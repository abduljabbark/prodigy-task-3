    HTML is the basic building block of a website. It organizes content like text, images, and buttons. Think of it as the framework or skeleton of a webpage.

    CSS makes the website look nice. It adds colors, styles, and layout to the content. It helps make the website look good on different devices like phones and computers.

  • bootstrap

    Bootstrap is a tool that gives you ready-made design templates like buttons and layouts. It helps you build websites faster and makes sure they look good on all screen sizes.

  • Java Script

    JavaScript makes a website interactive. It allows things like buttons to respond when clicked and enables more complex features like animations and pop-ups.
